# Prompt All-in-One YAML Editor 🎨

Stable Diffusion の拡張機能「[sd-webui-prompt-all-in-one](https://github.com/Physton/sd-webui-prompt-all-in-one)」で使用される `prepend.yaml` を、直感的な操作で編集・管理するための高機能Webエディタです。

---

## 🌟 主な機能

### 1. 直感的なドラッグ＆ドロップ操作
- **タグの移動**: 右側のタグカードを掴んで、左側のサイドバーにある別の「グループ」へ直接ドロップして移動できます。
- **並び替え**: カテゴリ、グループ、タグの順番を自由に入れ替えることが可能です。
- **検索結果からの移動**: 全体検索で抽出したタグを、そのまま別のグループへ引っ越しさせることもできます。

### 2. 強力なバリデーション（重複チェック）
- **自動連番**: 新しいタグを追加する際、`new_prompt`, `new_prompt_1` のように自動で重複しない名前を付与します。
- **保存時チェック**: 編集時や移動時に、同じグループ内に同一のプロンプト名が存在する場合、エラーを表示してデータの破壊を防ぎます。

### 3. スマート検索
- **ローカル検索**: 現在選択中のグループ内からタグを探します。
- **全体検索**: ファイル内のすべてのカテゴリ・グループを横断して検索します。検索結果には「どのカテゴリのどのグループに属しているか」のパスが表示されます。

### 4. 翻訳・カラーカスタマイズ
- **ワンクリック翻訳**: プロンプト名を入力して翻訳ボタンを押すと、自動的に日本語ラベルを生成します（MyMemory API使用）。
- **グループカラー**: グループごとに背景色をRGBA形式で細かく設定でき、視認性を高められます。

## 🚀 使い方

### 1. YAMLファイルを読み込む
- 起動時に `public/prepend.yaml` が自動で読み込まれます。
- 別のファイルを編集したい場合は、「開く」ボタンを押すか、画面に直接ファイルを **ドラッグ＆ドロップ** してください。

### 2. タグを編集する
- **編集**: タグカードを **ダブルクリック** すると編集モードに入ります。
- **保存**: 編集が終わったら「保存」ボタンを押してください。※重複がある場合は警告が出ます。

### 3. 整理・並び替え
- 各要素の右上にある **「⋮⋮」アイコン（グリップ）** をドラッグして移動させます。
- タグを別のグループに入れたい時は、左側のサイドバーにあるグループ名の上まで運んで離してください。

### 4. 成果物を保存する
- 左上の「保存」ボタンを押すと、編集済みの `prepend.yaml` がダウンロードされます。
- これを Stable Diffusion の拡張機能フォルダ（`extensions/sd-webui-prompt-all-in-one/tags/` など）に上書きして使用してください。
